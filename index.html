Progressive Scraper
Deployed GitHub more_vert
closeProgressive Scraper

Progressive Scraper is a web scraper that displays ten of the most recent progressive rock album reviews from Prog Archives. Users can save reviews to their favorites list and leave comments on their saved reviews. Comments can be individually deleted, and articles can be deleted in batches. This is to fully exemplify the CRUD functionality.

This app uses Express in conjunction with Node.js to create the server. Cheerio and Axios scrapes the site and the targeted elements are stored in a Mongo database using Mongoose. Handlebars.js is the view engine and renders the JSON responses from the server. Bootstrap supplies most of the CSS. This application follows the MVC design pattern.

Technologies used: MongoDB, Mongoose, Cheerio, Node.js, Express, Handlebars.js, jQuery, Bootstrap

Cinemaphile
Deployed GitHub more_vert
closeCinemaphile

Co-developed with 3 other team members, Cinemaphile is a dynamic web application that allows users to search for movies and store them in lists they can refer to. Cinemaphile uses Node.js and Express.js for the server and MySQL with Sequelize for data storage. Passport.js authenticates user login credentials. This application follows the MVC design pattern.

My contributions to the project include building the Express server and routes that respond to get, post, put, and delete requests. I implemented Handlebars.js as our view engine. I also integrated Passport.js for log-in validation.

Technologies used: MySQL, Sequelize, Node.js, Express.js, Handlebars.js, Passport.js, jQuery, and Bootstrap
